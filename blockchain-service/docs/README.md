## 系统简介

### 项目背景

EduChain 区块链存证服务是基于区块链存证的教育知识共享与智能检索系统的核心组件，为知识内容提供不可篡改的存证服务。

### 核心功能

- **内容存证**：将知识内容的哈希值记录到区块链
- **内容验证**：验证内容是否被篡改
- **证书生成**：生成PDF格式的存证证书
- **二维码验证**：支持扫码验证证书真伪

### 技术栈

| 技术 | 说明 |
|------|------|
| Python 3.11+ | 开发语言 |
| FastAPI | Web框架 |
| SQLAlchemy | ORM框架 |
| ReportLab | PDF生成 |
| qrcode | 二维码生成 |

### 系统特点

1. **轻量级**：不需要复杂的共识机制
2. **即时确认**：交易立即打包成区块
3. **证书可视化**：生成PDF证书
4. **易于部署**：支持Docker一键部署

## 快速开始

### 1. 启动服务

```bash
cd blockchain-service
pip install -r requirements.txt
python main.py
```

### 2. 访问API文档

浏览器打开：http://localhost:8000/docs

### 3. 测试存证功能

```bash
curl -X POST "http://localhost:8000/api/blockchain/certify" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "KNOWLEDGE_CERT",
    "knowledge_id": 1,
    "user_id": 1,
    "content_hash": "e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855"
  }'
```
