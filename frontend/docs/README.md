## 快速开始

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
# 标准开发模式
npm run dev

# Mock模式（无需后端）
npm run dev:mock
```

### 3. 访问应用

浏览器打开：http://localhost:3000

### 4. 构建部署

```bash
# 生产构建
npm run build

# 预览构建结果
npm run preview
```
